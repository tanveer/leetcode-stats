Leetcode-stats

A python script to get your problem stats on leetcode.

Motivation

I wanted to get more diversified stats of those problems I have worked on Leetcode. Due to NOT familiar with javascript/AJAX hacking, I built this python script with Selenium and PhantomJS to parse the problem tables.

Prerequisite

  • Selenium
pip install selenium
  • PhantomJS

Download the package on PhantomJS' website. Put the phantomjs binary in your path.
